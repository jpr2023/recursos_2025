Apps: el término "apps" es una abreviatura de "aplicaciones". Se refiere a un tipo específico de programa informático diseñado para realizar una o varias tareas específicas para el usuario final.

Clickear: Es la acción de presionar y soltar rápidamente uno de los botones de un ratón (mouse) o un dispositivo de entrada similar (como un trackpad o un botón en una tableta gráfica). Generalmente, se refiere al botón izquierdo del ratón, que es el botón principal utilizado para la mayoría de las interacciones.

Cursor o Puntero: Es un indicador visual en la pantalla de un ordenador que señala la posición actual donde el usuario está interactuando o donde la siguiente acción tendrá lugar. Su apariencia puede variar dependiendo del contexto y del sistema operativo o la aplicación que se esté utilizando.

Herramientas: Se refiere a una amplia variedad de programas de software, utilidades o incluso componentes de hardware diseñados para ayudar a los usuarios, desarrolladores o administradores de sistemas a realizar tareas específicas de manera más eficiente, automatizada o con mayor precisión dentro del ámbito de la computación.

interfaz: Una interfaz es el punto de conexión o el medio a través del cual dos o más componentes, sistemas o usuarios intercambian información o interactúan entre sí. Actúa como una capa de comunicación que permite que diferentes entidades trabajen juntas, incluso si tienen diseños o funcionalidades internas muy distintas.

Internet: Es una red global de redes de computadoras interconectadas que utilizan un conjunto de protocolos de comunicación estandarizados (principalmente TCP/IP) para permitir el intercambio de información entre miles de millones de dispositivos en todo el mundo.

Mouse: Dispositivo periférico de entrada que permite al usuario interactuar con la interfaz gráfica de un ordenador moviendo un cursor o puntero en la pantalla.

Navegar: Se refiere a la acción de moverse o desplazarse a través de diferentes recursos de información digital interconectados. Estos recursos pueden estar ubicados en la World Wide Web (internet), en un sistema de archivos local de una computadora, en una red local (LAN), o incluso dentro de una aplicación de software.

Pantalla: Una pantalla, también conocida como monitor o display, es un periférico de salida fundamental que permite al usuario visualizar la información procesada por la computadora. Actúa como una interfaz visual entre el sistema informático y el usuario, mostrando textos,

Periférico: Un periférico es cualquier dispositivo auxiliar e independiente que se conecta a la unidad central de procesamiento (CPU) de una computadora para ampliar sus funcionalidades. Estos dispositivos permiten la entrada de información al sistema, la salida de información desde el sistema, o ambas cosas. También existen periféricos de almacenamiento y de comunicación.

Programas o Software: Un programa en informática, también conocido como aplicación o software, es una secuencia de instrucciones escritas en un lenguaje de programación que le indica a una computadora qué tareas debe realizar y cómo debe realizarlas. Estas instrucciones son interpretadas o ejecutadas por el procesador (CPU) del ordenador para llevar a cabo una función específica o un conjunto de funciones.

Teclado: Es un periférico de entrada fundamental que permite al usuario introducir datos, comandos e instrucciones a una computadora o a otros dispositivos digitales (como tablets, smartphones, etc.) a través de la pulsación de teclas
